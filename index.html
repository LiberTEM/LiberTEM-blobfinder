LiberTEM-blobfinder¶

LiberTEM-blobfinder is a package for correlation-based peak finding and strain mapping. It is part of the LiberTEM project.

Usage¶

See Usage documentation!

Installation¶

The quick version from PyPI:

$ pip install libertem-blobfinder[pyfftw]

See full installation instructions for more details!
